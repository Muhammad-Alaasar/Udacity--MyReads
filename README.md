# My Reads Project
##### by _Muhammad Al-Aasar_


> This app allows you to select and categorize books you have read, are currently reading, or want to read.
#
### Technologies
###
- HTML (JSX)
- React JS
- JavaScript
- CSS

### Usage
###
Download or Clone it and do this steps:-
#####
First Step:
```sh
npm install
```
OR
```sh
npm i
```
Second Step:
```sh
127.0.0.1:3000
```
